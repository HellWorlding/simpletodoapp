# Todo App

간단한 할 일 관리 애플리케이션입니다.

## 기능

- 할 일 추가
- 할 일 완료 체크
- 할 일 삭제
- 로컬 스토리지를 통한 데이터 저장

## 파일 구조

```
todo-app/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
├── README.md
└── .gitignore
```

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 입력창에 할 일 입력
3. "추가" 버튼 클릭 또는 Enter 키 입력

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
