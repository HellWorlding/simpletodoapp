# Daily Todo Manager

간단하고 직관적인 할 일 관리 앱

## 기능

- 할 일 추가/삭제/수정 (더블클릭)
- 완료 체크 토글
- 카테고리별 분류 (일반, 업무, 개인, 쇼핑)
- 카테고리 필터링
- 진행률 대시보드
- 다크 모드 지원
- 모바일 반응형

## 사용 기술

- HTML5, CSS3, JavaScript (Vanilla)
- LocalStorage (데이터 및 테마 저장)
- CSS Variables (다크 모드)
